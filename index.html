• git config: Configuração de valores específicos do usuário, como e-mail e nome de usuário.
  • git init: Inicializa um novo repositório GIT.
  • git add: Adiciona arquivos ao índice.
  • git clone: Clona repositórios.
  • git commit: Confirma alterações.
  • git status: Exibe status dos arquivos.
  • git push: Envia alterações para o repositório remoto.
  • git checkout: Cria ou muda para um ramo.
  • git remote: Conecta-se a um repositório remoto.
  • git branch: Lista, cria ou exclui ramos.
  • git pull: Mescla alterações do repositório remoto para o local.
  • git merge: Mescla ramificações.
  • git diff: Lista conflitos.
  • git tag: Marca compromissos com etiquetas.
  • git log: Exibe lista de compromissos.
  • git reset: Redefine índice e diretório de trabalho para o último commit.
  • git rm: Remove arquivos.
  • git stash: Salva mudanças temporariamente.
  • git show: Visualiza informações de um objeto.
  • git fetch: Obtém objetos do repositório remoto.
  • git ls-tree: Exibe um objeto de árvore.
  • git cat-file: Exibe tipo de um objeto.
  • git grep: Procura frases ou palavras em arquivos.
  • gitk: Interface gráfica para repositório local.
  • git instaweb: Roda um servidor web em interface com repositório local.
  • git gc: Otimiza o repositório.
  • git archive: Cria arquivo zip ou tar de uma árvore do repositório.
  • git prune: Exclui objetos sem ponteiros de entrada.
  • git fsck: Verifica a integridade do sistema de arquivos.
  • git rebase: Reaplica compromissos em outro ramo.
